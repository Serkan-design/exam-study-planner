# 📚 Exam Study Planner

A modern web application that helps students create a smart and organized study schedule for upcoming exams.

The application calculates how many units you should study each day based on the exam date and total unit count, helping you stay consistent and prepared.

---

## 🌐 Live Demo

🔗 https://serkan-design.github.io/exam-study-planner/

---

## ✨ Features

✔️ Course based study planning
✔️ Automatic daily unit calculation
✔️ Exam countdown logic
✔️ Clean and modern dark UI
✔️ Mobile responsive design
✔️ Data persistence with LocalStorage
